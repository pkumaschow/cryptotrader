Written in python3 to run on the raspberry pi 4 (pihole.homelab.com) which is connected directly to my wifi router and online 24/7.
Source code to be stored in gitlab and use a pipeline to test and deploy to the raspbery pi4.

A TUI is required to monitor cryptocurrency rates for a specified list of currencies. bitcoin and ethereum initially.
Use Kraken for exchange. Funds will be kept there.
Continously monitor currencies and set triggers for buy and sell events.
A test mode is required to evaluate potential trades and produce some statistics to evaluate effectiveness and potential earnings.
Once happy with performance change the configuration to production mode to allow real trades.
